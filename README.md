# landingPage
This project is part of a web development learning exercise focused on building a complete landing page using **HTML** and **CSS**. The goal is to convert a provided design into a functional webpage while practicing core front-end development skills.
---
**Live Link:** https://scribencode.github.io/landingPage/
---
**Link to design:** https://cdn.statically.io/gh/TheOdinProject/curriculum/81a5d553f4073e593d23a6ab00d50eef8620796d/foundations/html_css/project/imgs/01.png
##Page Sections
- **Hero Section** – main headline (hero text)  
- **Section Two** – feature or content layout using flexbox  
- **Section Three** – additional information/images  
- **Section Four** – call-to-action or highlighted message  
- **Footer** – simple closing section
---

## Technologies Used

| Technology | Purpose |
|-----------|---------|
| HTML5     | Markup & structure |
| CSS3      | Styling & layout |
| Flexbox   | Handling content alignment |
---
##  Learning Objectives

This project improves understanding of:

- Structuring a web page with semantic tags  
- Styling visually based on design requirements  
- Flexbox layout techniques and spacing  
- Working iteratively (one section at a time)  
- Managing a simple project on GitHub
